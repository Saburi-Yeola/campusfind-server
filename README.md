# CampusFind – Server (Railway)

🚀 Fully functional, production-ready backend API for the CampusFind Lost & Found Portal.

## 🛠️ Features
- Node.js & Express
- MySQL Database (TypeORM/Pool)
- JWT Authentication
- Socket.io Real-time Chat
- Cloudinary Image Uploads
- Email Notifications

## 🌐 Local Setup
1. Configure `.env` with local MySQL credentials.
2. Run `npm install`.
3. Start with `node index.js`.
